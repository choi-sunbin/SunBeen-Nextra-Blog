영어 문서 : https://man7.org/linux/man-pages/man3/bzero.3.html <br />
- <strong>DESCRIPTION</strong>

```txt
bzero() 함수는 s가 가리키는 위치에서 시작하는 n 바이트 메모리에
```

### ft_memchr
영어 문서 : https://man7.org/linux/man-pages/man3/memchr.3.html
 
- <strong>DESCRIPTION</strong>

<summary>ft_memmove</summary>
### ft_memmove

영어 문서 : https://man7.org/linux/man-pages/man3/memmove.3.html
- <strong>DESCRIPTION</strong>
```txt
 memmove() 함수는 메모리 영역 src에서 메모리 영역 dest로 n개 바이트를 복사한다.
 메모리 영역이 겹칠 수 있다. 
 마치 src의 바이트들을 src나 dest와 겹치지 않는 임시 배열로 복사한 다음 
 그 임시 배열을 dest로 복사하는 것처럼 복사가 이뤄진다.
```
- <strong>RETURN VALUE</strong>
```txt

memmove() 함수는 dest 포인터를 반환한다.
 ```
- <strong>CODE</strong>

 ```c copy
 /* ************************************************************************** */
/*                                                                            */
/*                                                        :::      ::::::::   */
/*   ft_memmove.c                                       :+:      :+:    :+:   */
/*                                                    +:+ +:+         +:+     */
/*   By: sunbchoi <sunbchoi@student.42seoul.kr>     +#+  +:+       +#+        */
/*                                                +#+#+#+#+#+   +#+           */
/*   Created: 2020/12/23 21:33:08 by sunbchoi          #+#    #+#             */
/*   Updated: 2021/01/03 20:21:09 by sunbchoi         ###   ########.fr       */
/*                                                                            */
/* ************************************************************************** */

#include "libft.h"

void	*ft_memmove(void *dest, const void *src, size_t size)
{
	unsigned char		*temp_d;
	const unsigned char	*temp_s;

	if (dest == 0 && src == 0)
		return (0);
	if (dest <= src)
	{
		temp_d = (unsigned char*)dest;
		temp_s = (unsigned char*)src;
		while (size--)
			*temp_d++ = *temp_s++;
	}
	else
	{
		temp_d = (unsigned char*)dest + size;
		temp_s = (unsigned char*)src + size;
		while (size--)
			*--temp_d = *--temp_s;
	}
	return (dest);
}
 ```

 ```c copy
size_t	ft_strlcpy(char *dest, const char *src, size_t dest_size)

		return (0);

	else
	{
		ft_memcpy(dest, src, dest_size - 1);
		dest[dest_size - 1] = 0;
	}
	return (src_size);
}
 ```

 ```c copy
size_t		ft_strlcat(char *dest, const char *src, size_t size)
{
	size_t	loop;
	size_t	dest_len;
	size_t	src_len;

	dest_len = ft_strlen(dest);
	src_len = ft_strlen(src);
	loop = 0;
	if (dest_len > size - 1 || size == 0)
		return (size + src_len);
	if (dest_len < size - 1)
	{
		while (src[loop] != 0 && dest_len + loop < size - 1)
		{
			dest[dest_len + loop] = src[loop];
			loop++;
		}
	}
	dest[dest_len + loop] = 0;
	return (dest_len + src_len);
}
 ```
